# TextVibe Main

React/MUI user app for TextVibe. Set `VITE_API_BASE_URL` to the Django API base URL, or it defaults to `http://127.0.0.1:8000`.

Set `VITE_GOOGLE_CLIENT_ID` to a Google OAuth Web client ID if you want the login page to offer Google account selection before sending the OTP.
